#  Portafolio de Fundamentos de Python: REQ 1 - 6

Este repositorio contiene los primeros **seis scripts** diseñados para demostrar el dominio de los **Fundamentos de Programación en Python** a través de casos de uso reales de nuestra institución educativa internacional online "EDUNIDO".

##  Requerimientos Implementados

| REQ | Archivo | Concepto Principal |
| :--- | :------ | :--- |
| **1** | `req_1_ponderacion_entrega.py` | Variables y Operadores |
| **2** | `req_2_registro_estudiante.py` | Tipos de Datos (String/Int/Bool) |
| **3** | `req_3_clasificador_certificado.py` | Sentencias Condicionales (`if`/`elif`/`else`) |
| **4** | `req_4_seguimiento_modulos.py` | Sentencias Iterativas (`for`/`while`) |
| **5** | `req_5_directorio_cursos.py` | Estructuras de Datos (Listas/Diccionarios) |
| **6** | `req_6_calculadora_matricula.py` | Funciones (`def`) para Reutilización |
